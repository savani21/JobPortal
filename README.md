# Job Portal System

A full-stack Job Portal web application developed using the MERN Stack.

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- JavaScript
- HTML
- CSS

## Features
- User Authentication
- Admin Authentication
- Job Posting & Management
- Job Applications
- Profile Management
- CRUD Operations
- Responsive User Interface

## Installation

1. Clone the repository
2. Install dependencies using npm install
3. Configure environment variables
4. Run frontend and backend servers
